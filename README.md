## Installation

Clone the repository:

```bash
git clone https://github.com/raoufb3n/ozone-app.git
```
Navigate to the project directory:
```bash
cd ozone-app
```
Install dependencies:
```bash
flutter pub get
```
Run the app:
```bash
flutter run
```
